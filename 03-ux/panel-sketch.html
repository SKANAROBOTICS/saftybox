UUV Safety Box — Home Unit

Concept sketch for evaluation · independent motor-cutoff watchdog (secondary to PX4)

What each control does

  • LEASE / REMAINING — lease that will be granted next / time left, counting down.
  • CHANGE LEASE — guarded toggle, disarmed only: flick to cycle the length, hold to lock.
  • SINGLE — guarded toggle: grants one autonomy window of the PRESET length.
  • AUTO — guarded latched toggle: continuous heartbeat while on; slam the cover shut to disarm.
  • UN-ARM (mushroom) — press anytime to cut the motors; release never re-arms.
  • MASTER KEY — enables the whole box; with no key, nothing can arm.

The promise: the UUV's motors run only while a valid, live lease is held. Lose the comms link, hit the mushroom, or pull the key — and the motors stop. AUTO is for live operation (stops within seconds of comms loss); SINGLE grants a fixed autonomy window when the link will be down.
