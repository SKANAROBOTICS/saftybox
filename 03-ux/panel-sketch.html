UUV Safety Box — Home Unit

Concept sketch for evaluation · independent motor-cutoff watchdog (secondary to PX4)

Controls

ON / OFFPower
MODE — SAFEDisarmed
MODE — SINGLESend one lease
MODE — AUTOSend leases continuously
PROGRAM UPLonger lease (×2)
PROGRAM DOWNShorter lease (÷2)
UN-ARMHold: cut motors  ·  Release: resume

Screen & Indicators

LEASERun time granted to robot
REMAININGTime left on lease (may be slightly less)
ROBOTBlinks on each heartbeat · green: running · red: stopped
LINKSolid: connected · blink: weak · off: lost
ARMEDReady to send leases · blinks on each grant sent
